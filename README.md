# swot-analysis-website
ROLE:
You are a web developer.

TASK:
Create a simple educational single-page website that explains SWOT analysis to leaders who are evaluating AI adoption.

TECH:
Use HTML and CSS only (optional small JavaScript if needed).

DESIGN STYLE:
- Colorful design
- Basketball theme
- Modern and clean
- All content centered on the page
- Responsive for desktop and mobile
- Use cards, grids, and clear spacing
- Add simple hover effects

PAGE STRUCTURE:
The website must have exactly 5 sections stacked vertically.

SECTION 1: HERO
Title: "Using SWOT Analysis to Evaluate AI Adoption"

Add a short explanation that SWOT stands for:
Strengths, Weaknesses, Opportunities, and Threats.

Explain that leaders use SWOT analysis to evaluate important decisions like adopting artificial intelligence.

Include a basketball analogy:
"Just like a basketball coach studies their team before a big game, leaders analyze their organization before adopting AI."

SECTION 2: WHAT IS SWOT
Create four cards explaining:

Strengths – internal advantages  
Weaknesses – internal challenges  
Opportunities – external trends to take advantage of  
Threats – external risks like competitors or regulation

Add a basketball analogy below the cards:
Strengths = star players  
Weaknesses = team weaknesses  
Opportunities = open scoring chances  
Threats = strong opponents

SECTION 3: WHY LEADERS USE SWOT FOR AI
Explain why SWOT helps leaders decide whether to adopt AI.

Include bullet points:
- AI requires investment
- AI changes how employees work
- AI can improve productivity
- AI can create competitive advantage

Include a basketball analogy about adding a new play to the playbook.

SECTION 4: EXAMPLE SWOT FOR AI ADOPTION
Create a visual 2x2 SWOT grid.

Strengths
- Large internal data
- Strong tech infrastructure
- Leadership support

Weaknesses
- Limited AI expertise
- High cost
- Data quality issues

Opportunities
- Automate tasks
- Better customer insights
- Higher productivity

Threats
- Competitors adopting AI
- Regulation risks
- Cybersecurity issues

SECTION 5: KEY TAKEAWAYS
Show 3–4 key lessons:

- SWOT helps leaders evaluate AI risks and opportunities
- AI adoption requires strategy and planning
- Understanding strengths and weaknesses improves decisions
- Strategic planning supports innovation

End with this message:
"Great leaders analyze the game before they make the play."

FINAL REQUIREMENTS:
- Center all content
- Use colorful sections
- Add hover effects to cards
- Use a grid layout where needed
- Generate the complete HTML and CSS code.
